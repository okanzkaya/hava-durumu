# Hava Durumu Mobil Uygulaması

Bu proje, Java dili kullanılarak Android Studio ortamında geliştirilmiş basit bir hava durumu uygulamasıdır.

## Özellikler
- Şehir bazlı hava durumu sorgulama
- Anlık sıcaklık ve rüzgâr bilgisi görüntüleme
- Sade ve kullanıcı dostu arayüz

## Kullanılan Teknolojiler
- Java
- Android Studio
- Retrofit
- Open-Meteo API

## Kurulum
- Proje Android Studio ile açılır
- Uygulama emülatör veya gerçek cihazda çalıştırılabilir
